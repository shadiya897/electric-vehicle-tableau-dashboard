# Electric Vehicle Population Dashboard

This project presents an interactive Tableau dashboard analyzing electric vehicle population data.

## Project Overview
The dashboard provides insights into the distribution and trends of electric vehicles using data visualization techniques.

## Key Features
- Electric vehicles distribution by manufacturer
- Growth trend of electric vehicles over model years
- Relationship between electric range and vehicle price
- Top 10 electric vehicle models by driving range

## Interactive Filters
Users can explore the dashboard using filters for:
- Make
- Model Year
- Electric Vehicle Type

## Tools Used
- Tableau
- Data Visualization
- Data Analysis

## Dataset
Electric Vehicle Population Dataset
